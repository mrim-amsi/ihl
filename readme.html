# ⚖️ IHL Learning Platform

An educational platform for learning International Humanitarian Law through interactive lessons and quizzes 📚✨

--- ## 📌 Overview The **IHL Learning Platform** is a web-based system designed to provide users with structured lessons and assessments in **International Humanitarian Law (IHL)**. It aims to simplify complex legal concepts and enhance learning through interactive content. --- ## ✨ Features * 📖 Structured Lessons on IHL topics * 🧠 Interactive Quizzes & Exams * 📊 Instant Results & Performance Tracking * 👤 User Authentication (Login / Register) * 📰 News & Updates related to IHL * 📱 Fully Responsive Design * 🔐 Admin Dashboard for managing content --- ## 🛠️ Tech Stack * 💻 Frontend: HTML, CSS, JavaScript * ⚙️ Backend: PHP (MVC Architecture) * 🔗 API Integration (if applicable) * 🗄️ Database: MySQL * 🎨 UI: Clean & Educational Design --- ## 🧠 System Modules * **Lessons Module:** تقديم الدروس بشكل منظم * **Quiz Module:** اختبارات تفاعلية مع تصحيح فوري * **Users Module:** إدارة المستخدمين * **Progress Tracking:** متابعة تقدم المستخدم * **Admin Panel:** إدارة الدروس والاختبارات --- ## 📸 Screenshots
